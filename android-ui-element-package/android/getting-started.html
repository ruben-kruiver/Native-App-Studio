Getting Started

Requirements

To start using the AT&T Android UI Elements, you will need to have the following software downloaded and installed:

  • Download and install the latest Android SDK. Ensure it is release 6 or newer.
  • Eclipse 3.4 or 3.5
  • The AT&T Android UI Elements zip file (atnt_android.version.7z)
  • The ADT plugin for eclipse is recommended.
  • Use the Android SDK and AVD Manager to download the Android 1.5 or 1.6 SDK.
Getting started with Android

We recommend visiting the developer guide which has many tutorials and demos.

Using the AT&T Android UI Elements

1. The package contains several projects

  • AttWidgetLibrary - Component library project
  • AttButtons - Demo of buttons
  • AttImageButtons - Demo of image buttons
  • AttCheckbox - Demo of the checkbox widget
  • AttRadioButton - Demo of the radio button widget
  • AttExpandingText - Demo of the expanding text field widget
  • AttCloseEditText - Demo of the close edit text widget
  • AttLabelEditText - Demo of the label edit text widget
  • AttSearchTextField - Demo of the search edit text field widget
  • AttSwitchControl - Demo of the switch control widget
  • AttImageToggleButton - Demo of the image toggle button widget
  • AttSegmentedTextToggleButton - Demo of the segmented toggle button widget
  • AttStaticTextToggleButton - Demo of the static text toggle button widget

2. Add the library project to your workspace

Menu File -> Import -> Import Existing Project into Workspace

3. Next, Finish

4. Create a new Android project

5. The component library should be selected as a dependency of any project you create. Access project properties in the Android section and add the AttWidgetLibrary as a dependency.

Layout Configuration

6. Please note that the xmlns:att attribute of a LinearLayout or RelativeLayout needs to point to the package containing your activity code. Example AT&T Android UI checkbox 

<?xml version="1.0" encoding="UTF-8"?>
<RelativeLayout 
        xmlns:android="http://schemas.android.com/apk/res/android"
	xmlns:att="http://schemas.android.com/apk/res/com.att.checkbox"
	android:layout_width="wrap_content" 
	android:layout_height="wrap_content"  >
                .
                . 
                .

</RelativeLayout>

7. Start coding. You could start adding your components to your layout or java classes.
